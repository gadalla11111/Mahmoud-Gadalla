## How to install (takes about a minute)

**Claude on the web or desktop app:**

1. Go to Settings, then Capabilities
2. Find the Skills section and click Upload skill
3. Choose a .skill file from this download
4. Repeat for each skill you want - install all 9 or just the ones you need

**Claude Code:** unzip a .skill file (it's a zip) into `~/.claude/skills/`

Skills need a paid Claude plan with code execution / skills enabled in Settings > Capabilities.

## The clever bit: teach it your brand once

The first time you use any of these skills, it asks six quick questions about your brand - voice, audience, platforms, CTAs, banned words. Claude then saves your answers as a Brand Profile (to its memory, or a brand-profile.md file in your project) automatically.

Every skill in this set reads that same profile. Answer once, and all nine skills write in your voice forever.

## Using the skills

Just ask naturally - "write a caption for this photo", "repurpose this blog post", "plan my October content", "analyse this CSV". Claude spots the right skill and applies it automatically. You can also name them: "use the hook-generator on this video idea".
