# Apphub Submission Pack

Copy-paste blocks for manually submitting each plugin to the Qoder apphub at https://qoder.com/account/apphub-publications

**Repo base URL:** https://github.com/gadalla11111/Mahmoud-Gadalla/tree/main/qoder-plugins-publish/plugins

## Field glossary

| Field | Source |
|---|---|
| Plugin ID | name slug |
| Display name | display (CN) / displayEn |
| Description | desc (CN) / descEn |
| Version | version |
| Author | author |
| Category | derived per-plugin |
| Tags | 3-5 keywords per plugin |
| Repository URL | per-plugin URL below |
| Icon | assets/icon.svg in each plugin dir |
| Screenshots | NOT INCLUDED - generate separately if required |

## Note on screenshots

The plugins currently ship only an SVG icon, no screenshots. If the apphub requires screenshots, generate them from each README rendered view (one cover + one skills table page) using any markdown-to-image tool, save them to screenshots/<plugin>.png, and reference locally + upload.
