# Marketing

Full-spectrum marketing toolkit covering copywriting, ad compliance, competitive tracking, campaign planning, social media trends, and SEO optimization.

> **Disclaimer:** This plugin assists professional workflows and does not replace professional advice. All outputs should be reviewed by qualified professionals before being used for decision-making. Ad compliance check results do not constitute legal advice.

## Capability Tiers

```
+--------------------------------------------------+
|  ALWAYS (standalone)                              |
|  * Multi-platform copywriting (7 platforms)       |
|  * Ad-law compliance scan (restricted terms       |
|    + industry-specific rules)                     |
|  * Competitor SWOT analysis & differentiation     |
|  * Campaign design (online / offline / hybrid)    |
|  * SEO content optimization (keywords +           |
|    structure + meta)                              |
|  * Social media trend screening & newsjacking     |
+--------------------------------------------------+
```

## Target Roles

- **Content Manager** -- Day-to-day copywriting, social media topic selection (recommended: Copywriting + Social Media Trends)
- **Brand Manager** -- Competitor monitoring and brand strategy (recommended: Competitive Tracking + Ad Compliance Check)
- **Marketing Director** -- Campaign planning and marketing oversight (recommended: Campaign Planning + Competitive Tracking)
- **Growth Manager** -- SEO and data-driven growth (recommended: SEO Content Optimization + Copywriting)

## Quick Commands

| Command | Description | Typical Input |
|---------|-------------|---------------|
| `/copywriting` | Multi-platform copy generation | Product/campaign info + target platform, e.g.: New product launch, Instagram sponsored post |
| `/ad-compliance-check` | Ad-law restricted-term & industry compliance scan | Paste the ad copy to check |
| `/competitive-tracking` | Competitor activity analysis & positioning comparison | Competitor names (1-5), e.g.: Nike, Adidas |
| `/campaign-planning` | Full campaign proposal design | Event goal + type + budget range, e.g.: Offline launch event, budget $15K |
| `/social-media-trends` | Cross-platform trend collection & newsjacking ideas | Optional: specify an industry or brand name |
| `/seo-content-optimization` | Article SEO analysis & optimization | Paste article content, or provide a URL for SEO diagnosis |
| `/marketing-performance` | Ad spend ROI analysis | Upload campaign data or key metrics |
| `/brand-voice-review` | Content-brand consistency audit | Brand description + content to review |

## Skills

| Skill | Description | Core Output |
|-------|-------------|-------------|
| Copywriting | Generate platform-adapted copy from product info | Full copy + 3 headline variants + A/B test recommendations |
| Ad Compliance Check | Scan for restricted terms against advertising regulations | Compliance report + revision suggestions + compliant copy |
| Competitive Tracking | Competitor activity analysis, positioning reverse-engineering, differentiation | Competitor report + feature matrix + battlecard |
| Campaign Planning | Design a complete campaign based on goals and budget | Campaign proposal + budget sheet + promotion plan |
| Social Media Trends | Cross-platform trend aggregation, newsjacking analysis, topic ideas | Trend report + topic suggestions + content calendar |
| SEO Content Optimization | Website SEO audit and article content optimization | SEO score report + optimized article |
| Marketing Performance | Marketing spend ROI analysis and channel efficiency evaluation | Performance report + funnel diagnosis + optimization checklist |
| Brand Voice Review | Audit content for brand-voice consistency | Review report + before/after comparison + channel adaptation |
