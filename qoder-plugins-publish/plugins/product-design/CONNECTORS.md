# 连接器

## 能力层级

**独立能力**（无连接器也完全可用）：
- 问题框定、设计简报、用户故事、用户旅程、视觉情绪板
- Web/Mobile 页面 Flow + 代码产出、异常态穷举、数据可视化
- 动效规划与落地、设计走查、无障碍审计、可用性测试
- 度量蓝图、工程 PRD、设计提取（外部站点 / 内部代码模式）、设计复盘
- 所有输出模板、方法论和决策框架完整可用

**增强能力**（连接工具后增强）：
- Figma → 从设计稿抽 design tokens、读 frame 作上下文、QA 对比设计 vs 实现
- designlang → 从线上站点反向抽 19 类 tokens + 设计质量评分
- Notion / 飞书文档 → Brief / Journey / PRD / Pitch / Retro 一键写入团队 wiki
- Linear / Jira → Stories 拆 issue、PRD 建 ticket、Metric 拉 sprint 指标
- GitHub → Extract 读远程仓库 tokens、Flow Web/Mobile 生成代码开 PR

## 连接器清单

| 连接器 | 增强能力 |
|--------|---------|
| **Figma** | 读设计稿 frame / Variables / Styles 作上下文；Extract Figma 模式抽 tokens；QA / Check 对比设计 vs 实现 |
| **designlang** | 从线上站点反向抽 19 类 design tokens + 设计质量评分 |
| **Notion / 飞书文档** | Brief / Journey / PRD / Pitch / Retro 等核心交付物一键写入团队 wiki，搜索历史文档作参考 |
| **Linear / Jira** | Stories 自动拆 issue / PRD 建 ticket / Retro 拉迭代数据 / Metric 拉 sprint 指标 |
| **GitHub** | Extract 内部模式读远程仓库 tokens；Flow Web/Mobile 生成代码开 PR；QA 对比 PR diff |
| **WebSearch** | 竞品实时信息获取、行业趋势参考 |

> 没有连接器也完全可以使用——以上功能会通过手动输入和本地文件输出来替代。
