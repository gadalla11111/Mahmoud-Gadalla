# AI 标题优化

## 功能

基于商品图片解析 + 原标题，由 LLM 生成更优秀的新标题。支持单商品与批量两种模式。

- 状态文件：`.title_state.json`（单商品）/ `.batch_title_state.json`（批量），应用后依赖该文件
- 仅解析第一张主图，避免多图超时
- 批量建议每批 ≤ 10 个商品（上下文串行拉取）

---

## 流程总览

```
拉取上下文 → Agent生成标题 → ⏸ 商家确认 → 应用标题
                                （⛔ 必须停下）    （AI标题 或 自定义标题）
```

---

## 单商品流程

**做什么：**
1. 拉取上下文（原标题 + 首图解析结果）：`ai_title_modify --offer_id <id>`
2. LLM 综合"原标题 + 图片文字 + 可见卖点 + 构图摘要"生成新标题。**若原标题与图片偏差过大，彻底抛弃原标题，以图片解析内容为准**
3. 暂存 AI 标题：`ai_title_modify --offer_id <id> --suggest "<title>"`
4. 将 CLI 返回的 markdown 展示给商家

**可用 API：** [商品查询](../apis/cbu_offer_query_tool.md) · [图片解析](../apis/cbu_image_analyse_tool.md)

---

### ⛔ 商家确认（必须停下）

展示 AI 推荐标题，询问商家选择：

1. 直接使用 AI 标题 → `ai_title_modify --offer_id <id> --apply`
2. 自定义标题 → `ai_title_modify --offer_id <id> --subject "<title>"`

> ⛔ 必须停下等商家回复。严禁未经确认自行应用标题。

---

## 批量流程

**做什么：**
1. 批量拉取上下文：`ai_title_modify --offer_ids <id1> <id2> ...`（空格分隔）
   - 查询/解析失败的商品标记后跳过，不中断整体
2. 逐个生成 AI 标题（解析失败的跳过）
3. 批量暂存：`ai_title_modify --batch-suggest '<JSON>'`
   - 格式：`'{"商品ID1":"新标题1","商品ID2":"新标题2"}'`
4. 表格化展示所有 AI 标题给商家

**可用 API：** [商品查询](../apis/cbu_offer_query_tool.md) · [图片解析](../apis/cbu_image_analyse_tool.md)

---

### ⛔ 商家确认（必须停下）

展示批量 AI 标题表格，询问商家选择：

1. 全部应用 AI 标题 → `ai_title_modify --batch-apply`
2. 部分调整 → `ai_title_modify --batch-apply '<JSON>'`（支持 AI + 自定义混合，格式同上）
3. 重新生成 → 覆盖暂存后重新展示

> ⛔ 必须停下等商家回复。严禁未经确认自行批量应用标题。

---

## 禁止事项

| 规则 | 说明 |
|------|------|
| 禁止跳过商家确认 | 生成标题后必须展示并等待商家选择，不得自行应用 |
| 禁止自行批量应用 | 批量生成后必须展示表格确认，不得自动应用全部 |
| 禁止使用多图解析 | 仅解析第一张主图，多图会超时 |
| 禁止忽略偏差 | 原标题与图片偏差过大时，必须抛弃原标题以图片为准 |

---

## 标题长度限制

- 上限：**30 字符**
- 计算规则：中文/全角符号 = **1** 字符，英文/数字/半角符号 = **0.5** 字符
- 生成要求：**尽可能写满 30 字符**，但**不得包含重复词语**
- 校验时机：暂存或提交时自动校验
- 超限行为：单商品直接报错；批量模式超限标题单独报错，不影响其他标题
- 错误示例：`标题长度超出限制：当前 35 字符（限制 30 字符），请缩短标题后重试。`

## 异常处理

| 异常 | 处理 |
|------|------|
| 商品查询失败 | 检查商品 ID 是否正确、是否属于当前账号 |
| 图片解析提交失败 | 重试，或检查商品是否有可用主图 |
| 解析轮询超时 | 30 秒超时后提示稍后重试 |
| 未找到 AI 标题 | 先拉取上下文并暂存 AI 标题再应用 |
| 未找到批量 AI 标题 | 先批量拉取上下文，再暂存批量 AI 标题 |
| 批量部分失败 | 返回成功/失败表格，失败项可用单商品自定义标题重试 |
| 标题长度超限 | 缩短标题后重新提交 |
| 标题修改失败 | 检查商品是否属于当前账号 |
