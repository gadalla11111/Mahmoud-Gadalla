# 科技服务助手

> 科技服务全流程工具，贯通从需求和供给两端流程闭环。需求端覆盖技术需求挖掘、成果搜索匹配、价值评估、对接方案生成、项目全周期跟进和政策资金匹配；供给端覆盖产品机会发现、产品定义规划、技术论证、原型验证、量产规划、商业化策略和融资准备。可支持连接益企新、浙企之家等服务后自动增强，自动链接全国科技金融、综合科技咨询、企业孵化、知识产权、研发设计、技术转移转化、检验检测、信息技术和知识产权等八大领域科技服务资源，也可纯手动输入使用。

## 适合谁

### 需求侧（技术引进方 / 技术经理人）


| 角色                          | 解决什么问题                                                     |
| ----------------------------- | ---------------------------------------------------------------- |
| **技术经理人**                | 系统化完成科技成果转化的全流程管理，提升服务标准化与专业输出质量 |
| **科技服务机构**              | 规范成果评价与对接流程，缩短项目周期                             |
| **高校/科研院所技术转移人员** | 规范成果评价、估值与对接流程                                     |
| **企业技术战略部门**          | 辅助技术引进决策与政策申报                                       |

### 供给侧（技术输出方 / 科技创业者）


| 角色              | 解决什么问题                               |
| ----------------- | ------------------------------------------ |
| **科技创业者**    | 系统化完成从机会识别到产品上市的全流程规划 |
| **产品经理**      | 规范产品定义、验证与商业化策略             |
| **硬件工程师**    | 结构化完成EVT/DVT/PVT验证与量产准备        |
| **投资人/孵化器** | 评估创业项目的研发成熟度与商业化可行性     |

## 何时不用

- 需要替代专业法律、财务或技术评估意见的场景
- 涉及国家秘密或涉密技术的转移评估
- 跨境技术转移中涉及出口管制清单的技术
- 融资输出不构成证券发行、投资邀约或FA服务
- 市场预测和估值仅为假设测算，不替代专业判断

## 工作流程

### 需求端 — 技术引进全流程

```
Step 1: /技术需求挖掘
  输入：企业名称或行业关键词
  输出：需求画像（痛点排序、学科映射）、技术需求清单、建议对接资源

Step 2: /成果搜索与匹配
  输入：技术需求描述 + 成熟度期望
  输出：匹配度排序的科技成果/专利/团队推荐清单 + 技术路线对比

Step 3: /技术价值评估
  输入：专利号或技术方案描述
  输出：商业化可行性评估报告（先进性、市场空间、风险、定价参考）

Step 4: /对接方案生成
  输入：供方成果信息 + 需方企业信息
  输出：技术转移合作方案书（模式、交易结构、里程碑、风控）

Step 5: /项目跟进管理
  输入：项目名称 + 当前阶段 + 卡点
  输出：Pipeline看板 + 推进建议 + 风险预警

Step 6: /政策与资金匹配
  输入：企业信息 + 技术方向
  输出：可申报政策清单 + 符合性判定 + 申报材料框架
```

### 供给端 — 产品研发与商业化全流程

```
Step 1: /产品机会发现
  输入：技术方向/行业
  输出：机会分析报告（痛点地图、市场空白、竞品盲区、差异化切入点）

Step 2: /产品定义与规划
  输入：产品机会描述
  输出：PRD文档、MVP范围、产品路线图

Step 3: /技术方案论证
  输入：产品定义 + 技术约束
  输出：技术可行性报告（路线对比、BOM估算、风险矩阵）

Step 4: /原型验证规划
  输入：技术方案 + 待验证假设
  输出：EVT计划书（实验设计、评判标准、迭代策略）

Step 5: /设计验证与量产规划
  输入：EVT结果 + 迭代方向
  输出：DVT/PVT清单、量产准备度评估、认证路径

Step 6: /商业化策略
  输入：产品定位 + 目标市场
  输出：GTM策略文档（定价、渠道、种子用户、PMF指标）

Step 7: /融资准备
  输入：公司阶段 + 业务数据 + 融资目标
  输出：BP框架、数据房清单、估值参考、投资人匹配
```

### 供需两侧协同

```
需求侧链路：
  /技术需求挖掘 → /成果搜索与匹配 → /技术价值评估
  → /对接方案生成 → /项目跟进管理 → /政策与资金匹配

供给侧链路：
  /产品机会发现 → /产品定义与规划 → /技术方案论证
  → /原型验证规划 → /设计验证与量产规划
  → /商业化策略 → /融资准备

协同闭环：
  1. 需求侧输出"技术需求描述"供供给侧优化产品定义
  2. 供给侧输出"成果信息"与"商业化可行性"供需求侧完善对接方案
  3. 需求侧"估值区间"与供给侧"BOM成本"双向参考，形成合理定价
  4. /项目跟进管理 同步跟踪供需双方项目推进状态
```

## 完整案例

### 案例A：需求侧 — 固态电池技术转移全流程

```
Step 1: /技术需求挖掘 "绿能动力科技，主营三元锂电池，计划布局固态电池"
  → 输出：需求画像（材料/装备/人才瓶颈）、技术需求清单、学科映射

Step 2: /成果搜索与匹配 "硫化物固态电解质，离子电导率≥10 mS/cm，TRL≥5"
  → 输出：Top 5 推荐成果（华科专利匹配度92分）、技术路线对比矩阵

Step 3: /技术价值评估 "专利号 ZL202210XXXXXX.X"
  → 输出：技术先进性评估、TAM/SAM/SOM测算、定价参考（3000-5000万）

Step 4: /对接方案生成 "供方：华科... 需方：绿能动力..."
  → 输出：合作方案书（独占许可+合作开发，入门费800万，里程碑付款）

Step 5: /项目跟进管理 "项目：固态电池技术转移，阶段：对接阶段40%，卡点：合作模式未定"
  → 输出：Pipeline更新、推进建议、风险预警

Step 6: /政策与资金匹配 "企业：绿能动力，技术方向：固态电池"
  → 输出：Top 5 可申报政策（国家重点研发计划、江苏省成果转化专项等）
```

### 案例B：供给侧 — AI质检一体机从0到1全流程

```
Step 1: /产品机会发现 "AI工业视觉，面向制造业质检"
  → 输出：用户痛点地图、市场空白（5万元以下AI质检一体机）、竞品盲区、差异化策略

Step 2: /产品定义与规划 "面向中小型制造企业的AI质检一体机，核心价值7天部署、漏检率<1%"
  → 输出：PRD（用户画像、功能分层P0/P1/P2、技术指标、MVP范围、路线图）

Step 3: /技术方案论证 "基于PRD，约束：成本<8000元，工期9个月"
  → 输出：系统架构、技术路线对比（RK3588 vs Jetson）、BOM估算（2540元）、风险矩阵

Step 4: /原型验证规划 "RK3588方案，假设：漏检率<1%、速度≥30件/分钟"
  → 输出：EVT计划书（假设分类P0/P1、实验设计、评判标准、迭代决策树）

Step 5: /设计验证与量产规划 "EVT通过，迭代：优化安装流程"
  → 输出：DVT/PVT清单、量产准备度评估（当前3.5分）、认证路径（CCC/CE）

Step 6: /商业化策略 "产品定位：中小企业AI质检，目标：长三角"
  → 输出：GTM策略（定价1.98万起、渠道、种子用户20家、PMF指标体系）

Step 7: /融资准备 "Pre-A轮，未产生收入，融资800万"
  → 输出：BP框架、数据房清单、估值参考（4000-5000万）、融资时间表16周
```
