# 融资准备方案报告模板

## 使用说明

本模板适用于公司启动融资前，系统梳理融资定位、BP框架、数据房准备、估值参考与投资人策略。请根据公司阶段（天使/Pre-A/A/B/C）裁剪使用。

### 2.2 各章节撰写要点与检查项

#### 第1章：项目概述

- [ ] 一句话定位清晰，无行业黑话
- [ ] 数据亮点3-4个，真实可验证
- [ ] 融资概要完整（金额/出让/用途）
- [ ] 视觉呈现简洁（建议信息图）

#### 第2章：市场机会

- [ ] 痛点描述具体，有场景支撑
- [ ] TAM/SAM/SOM有推导逻辑
- [ ] 趋势分析说明"为什么是现在"
- [ ] 数据来源标注

#### 第3章：产品与解决方案

- [ ] 产品展示有截图/流程图/实物图
- [ ] 用户旅程完整
- [ ] 技术壁垒具体（专利/算法/know-how）
- [ ] 竞品对比表格清晰

#### 第4章：商业模式

- [ ] 收入来源明确
- [ ] 成本结构合理
- [ ] 单位经济数据完整（客单价/毛利率/CAC/LTV）
- [ ] 盈利路径有时间表

#### 第5章：竞争格局

- [ ] 竞品分类完整（直接/间接/潜在）
- [ ] 对比客观，不贬低对手
- [ ] 壁垒分析有说服力

#### 第6章：运营数据

- [ ] 核心指标选择恰当（非虚荣指标）
- [ ] 趋势图直观展示增长
- [ ] 统计口径与截止时间标注
- [ ] 增长驱动因素分析到位

#### 第7章：团队介绍

- [ ] 核心成员履历与项目相关
- [ ] 团队分工完整
- [ ] 顾问/资源补充说明

#### 第8章：财务规划

- [ ] 历史数据真实
- [ ] 预测假设清晰
- [ ] 资金用途细化到方向
- [ ] 敏感性分析（如有）

#### 第9章：融资计划

- [ ] 出让比例合理
- [ ] 资金用途具体
- [ ] 里程碑对赌（如有）可达成
- [ ] 退出路径分析

#### 第10章：愿景与里程碑

- [ ] 愿景宏大但可信
- [ ] 里程碑具体可衡量
