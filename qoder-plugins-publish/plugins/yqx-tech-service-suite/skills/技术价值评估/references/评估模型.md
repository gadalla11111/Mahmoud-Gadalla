# 评估模型参考

## 一、技术先进性评估矩阵

### 1.1 评估维度与评分标准

技术先进性评估从四个维度进行量化评分，每个维度1-5分，总分20分。

| 维度 | 5分（领先） | 4分（先进） | 3分（中等） | 2分（跟随） | 1分（落后） |
|------|-------------|-------------|-------------|-------------|-------------|
| 技术创新度 | 原创性突破，开辟新方向 | 显著改进，解决行业难题 | 常规改进，有一定新意 | 微小改进，跟随主流 | 基本无创新，成熟技术复用 |
| 技术壁垒 | 完整专利池+深度know-how | 核心专利+一定know-how | 单件/少数专利保护 | 专利布局薄弱 | 无专利保护，易复制 |
| 技术成熟度 | TRL 8-9，已规模化应用 | TRL 6-7，原型/现场验证 | TRL 4-5，实验室/模拟验证 | TRL 2-3，概念/关键功能验证 | TRL 1，仅原理发现 |
| 团队持续创新 | 持续高产，产业化经验丰富 | 产出稳定，有产业化经历 | 有一定产出，经验一般 | 产出较少，缺乏产业化 | 团队不稳定，经验不足 |

### 1.2 先进性等级划分

| 总分区间 | 先进性等级 | 说明 |
|----------|------------|------|
| 17-20分 | 领先型 | 在细分领域具有显著技术优势 |
| 13-16分 | 先进型 | 技术具有竞争力，具备商业化基础 |
| 9-12分 | 中等型 | 技术成熟度或创新度有提升空间 |
| 5-8分 | 跟随型 | 技术同质化明显，差异化不足 |
| 4分以下 | 落后型 | 技术已接近淘汰或严重不成熟 |

## 二、专利价值评估模型

### 2.1 成本法

**基本原理**：以研发该技术所投入的成本为基础，考虑通货膨胀和贬值因素进行估值。

**适用场景**：
- 技术尚未产生收益，缺乏市场交易案例
- 研发成本记录完整、可追溯
- 作为估值下限参考

**公式**：
```
评估值 = 历史研发成本 × (1 + 合理利润率) × 成新率
```

**注意事项**：
- 成本法往往不能反映技术的未来收益潜力
- 成新率需结合技术迭代周期判断
- 建议与其他方法结合使用，不作为单一估值依据

### 2.2 市场法

**基本原理**：参照同类技术的近期交易价格，通过对比调整得出估值。

**适用场景**：
- 存在可查询的同类技术交易案例
- 技术所属行业交易活跃
- 作为市场接受度的参考

**操作步骤**：
1. 收集近期同类技术交易案例（专利转让、许可、作价入股）
2. 筛选可比案例：技术领域相似、交易时间相近、权利状态类似
3. 建立对比维度：技术成熟度、市场应用广度、剩余保护期、权利稳定性
4. 对各维度进行加权调整，得出估值区间

**注意事项**：
- 技术交易案例信息通常不公开，获取难度较大
- 每项专利具有独特性，完全可比案例较少
- 建议至少收集3个案例进行对比

### 2.3 收益法

**基本原理**：预测技术未来可产生的经济收益，按适当折现率折算为现值。

**适用场景**：
- 技术已进入市场或有明确商业化路径
- 未来收益可合理预测
- 作为估值上限参考

**简化模型**：
```
评估值 = Σ(预期年净收益 × 分成率) / (1 + 折现率)^n
```

**关键参数参考**：

| 参数 | 说明 | 参考范围 |
|------|------|----------|
| 分成率 | 技术收益占总收益的比例 | 行业惯例3%-7%，具体因行业而异 |
| 折现率 | 考虑风险的资金时间价值 | 通常10%-25%，技术风险越高折现率越高 |
| 收益期 | 可产生收益的时间长度 | 不超过专利剩余保护期或技术生命周期 |

**注意事项**：
- 收益预测存在较大不确定性，应进行敏感性分析
- 分成率需参考行业惯例，不可随意设定
- 建议给出乐观/中性/悲观三种情景的估值区间

### 2.4 综合评估建议

| 方法 | 适用阶段 | 估值性质 | 建议权重 |
|------|----------|----------|----------|
| 成本法 | 研发期/早期 | 下限参考 | 20%-30% |
| 市场法 | 成长期/成熟期 | 市场参考 | 30%-40% |
| 收益法 | 成熟期/产业化期 | 上限参考 | 30%-50% |

## 三、市场规模预测方法

### 3.1 TAM/SAM/SOM测算框架

| 指标 | 全称 | 定义 | 测算方法 |
|------|------|------|----------|
| TAM | Total Addressable Market | 总可及市场 | 技术理论上可应用的全部全球市场容量 |
| SAM | Serviceable Available Market | 可服务市场 | 基于企业资源与渠道可触达的目标区域/行业市场 |
| SOM | Serviceable Obtainable Market | 可获得市场 | 基于竞争格局短期内可实际获取的市场份额 |

### 3.2 TAM测算方法

**自上而下法**：
```
TAM = 目标行业全球总产值 × 该技术渗透率
```
- 行业总产值数据来源：行业统计年鉴、行业协会报告、上市公司财报汇总
- 渗透率参考：现有类似技术的渗透率、专家访谈、政策驱动强度

**自下而上法**：
```
TAM = 单客户平均价值 × 全球潜在客户数量
```
- 单客户价值：基于试点客户数据或行业均价推算
- 客户数量：基于行业企业名录、下游客户数量推算

### 3.3 SAM测算方法

```
SAM = TAM × 目标区域占比 × 目标细分行业占比 × 渠道可触达率
```

**调整因素**：
- 目标区域占比：如聚焦中国市场，则按中国市场占全球比例折算
- 细分行业占比：如技术仅适用于某行业中的特定细分
- 渠道可触达率：考虑现有销售/合作渠道覆盖能力

### 3.4 SOM测算方法

```
SOM = SAM × 预期市场占有率
```

**市场占有率预测参考**：

| 竞争态势 | 预期3-5年市场占有率 | 适用情景 |
|----------|---------------------|----------|
| 技术领先+先发优势 | 15%-30% | 新兴细分市场，竞争格局未定 |
| 技术先进+正面竞争 | 5%-15% | 已有竞争者，但有差异化优势 |
| 技术跟随+后发进入 | 1%-5% | 成熟市场，需逐步渗透 |
| 利基市场聚焦 | 20%-50% | 超细分领域，专注特定场景 |

### 3.5 增长驱动因素分析

| 驱动类型 | 具体因素 | 数据来源 |
|----------|----------|----------|
| 政策驱动 | 补贴政策、强制标准、进口替代政策、环保法规 | 政府公报、政策文件 |
| 需求驱动 | 下游行业增速、消费升级、替代需求、新兴市场 | 行业研究报告、统计数据 |
| 技术驱动 | 技术迭代周期、成本下降曲线、性能提升空间 | 技术路线图、历史数据趋势 |
| 供给驱动 | 产能扩张、供应链成熟、原材料降价 | 产业链调研、上市公司公告 |

## 四、竞品分析框架

### 4.1 竞品识别方法

1. **直接竞品**：解决相同技术问题、面向相同应用场景的技术方案
2. **间接竞品**：通过不同技术路径解决相同需求的技术方案
3. **替代威胁**：可能使该技术需求消失的新技术或新模式

### 4.2 竞品对比维度

| 对比维度 | 对比内容 | 信息来源 |
|----------|----------|----------|
| 核心性能指标 | 关键技术参数横向对比 | 产品手册、检测报告、论文数据 |
| 成本结构 | 制造成本、使用成本、维护成本 | 行业调研、供应链信息、公开报价 |
| 成熟度 | TRL等级、应用案例数量、用户反馈 | 公开报道、用户评测、行业口碑 |
| 专利壁垒 | 专利布局数量、核心专利稳定性 | 专利数据库、无效宣告记录 |
| 市场渗透率 | 市场份额、客户数量、营收规模 | 行业报告、上市公司财报 |
| 团队背景 | 核心团队技术背景、产业化经验 | 公开资料、LinkedIn、学术主页 |
| 商业模式 | 技术授权、产品销售、服务收费 | 官网、公开报道 |

### 4.3 竞品动态监控要点

- 竞品专利申请动态（新技术方向布局）
- 竞品融资/并购动态（资源整合能力变化）
- 竞品客户拓展动态（市场认可度变化）
- 行业标准制定参与情况（话语权变化）

## 五、定价参考方法

### 5.1 许可费率参考

| 行业领域 | 常见许可费率范围 | 计费基础 | 说明 |
|----------|------------------|----------|------|
| 医药/生物技术 | 2%-10% | 产品销售额 | 视研发阶段和专利重要性而定 |
| 电子/半导体 | 3%-7% | 产品销售额/使用费 | 标准必要专利费率通常有上限 |
| 软件/互联网 | 5%-15% | 营收/用户数 | SaaS模式按订阅收入比例 |
| 新材料 | 3%-6% | 产品销售额 | 视材料在终端产品中的价值占比 |
| 高端装备 | 2%-5% | 设备销售额 | 通常一次性付费或按台计费 |

### 5.2 转让对价参考因素

| 因素 | 对价影响方向 | 评估要点 |
|------|--------------|----------|
| 专利剩余保护期 | 正向 | 剩余保护期越长，对价越高 |
| 专利稳定性 | 正向 | 经历过无效宣告且维持有效的专利价值更高 |
| 同族专利数量 | 正向 | 全球同族布局广，对价相应提高 |
| 技术成熟度 | 正向 | TRL越高，产业化风险越低，对价越高 |
| 市场验证程度 | 正向 | 已有客户/收入验证的技术对价更高 |
| 卖方紧迫度 | 负向 | 卖方出售意愿强，对价可能降低 |

### 5.3 入股估值参考

**技术作价入股的一般原则**：

- 技术出资比例通常不超过注册资本的70%（以公司法及相关规定为准）
- 作价金额应经第三方评估机构评估
- 需考虑技术后续改进的权益归属

**常见作价方式**：

| 方式 | 说明 | 适用情景 |
|------|------|----------|
| 固定作价 | 一次性确定技术入股的金额 | 技术成熟，未来改进空间小 |
| 分期调整 | 根据里程碑达成情况调整作价 | 技术尚需验证，风险较高 |
| 收益挂钩 | 作价与后续实际收益挂钩 | 收益不确定性大，需绑定长期利益 |

### 5.4 定价声明要求

所有定价参考必须：
1. 明确标注参照来源（行业惯例、交易案例、评估模型）
2. 说明适用条件与限制
3. 给出区间而非单一数值
4. 声明不构成正式估值意见，建议委托专业机构出具正式报告
